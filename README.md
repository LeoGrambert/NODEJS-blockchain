# NODEJS-blockchain

Simple POC of a POW blockchain using Node.js

## Installation

```
npm install
```

## Launch

You can change proof of work difficulty in ./index.js with `DIFFICULTY_LEVEL` variable.<br>
The default difficulty is 4.

```
npm start
```
